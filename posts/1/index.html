1. 获取文件的大小

2020.08.01

在go语言中，我们有很多方法获取一个文件的大小。

Read字节方式

第一种，当然是最常见的read()方法，也就是打开文件，把文件读一遍

func main() {
    file, err := os.open("filename")
    if err == nil {
    	sum := 0
    	buf := make([]byte, 1024)
    	for {
        	n, err := file.read(buf)
        	sum += n
        	if err == io.EOF {
            	break
        	}
    	}
    	fmt.Println("file size is ", sum)   
    }
}

这种方式需要打开文件，通过for循环反复的读取文件里面的内容，然后去计算文件的大小。效率低，代码量大。

ioutil方式

由于上面的代码量比较大，所以我们可能立刻会想到使用ioutul包的ReadFile()方法来替代，直接获取文件的内容，然后计算出文件的大小

func main() {
    contents, err := ioutil.ReadFile("filename")
    if err == nil {
        fmt.Println("file size is ", len(contents))
    }
}
